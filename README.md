# 📊 Sales Performance Dashboard

## 1️⃣ Project Title

**Sales Performance Analysis using Excel**

---

## 2️⃣ Brief One-Line Summary

An interactive **Excel-based sales analysis dashboard** built using **Pivot Tables, Pivot Charts, and slicers** to track sales performance, profit trends, and product performance.

---

## 3️⃣ Overview

This project analyzes sales data to uncover business insights such as **regional performance, product sales trends, profit analysis, and monthly sales growth**.

The dashboard was built entirely in **Microsoft Excel** and provides a dynamic and user-friendly way to explore the data using **interactive slicers and charts**.

This project demonstrates practical **data analysis and visualization skills** used by Data Analysts to transform raw data into meaningful business insights.

---

## 4️⃣ Dataset

* **Source:** `sales_dashboard_dataset.xlsx`

* **Description:**
  The dataset contains sales transaction records including:

* Order ID

* Order Date

* Region

* Product

* Category

* Sales Amount

* Profit

* Quantity Sold

The dataset is used to analyze **sales performance across different regions, products, and time periods**.

---

## 5️⃣ Tools and Technologies

* **Microsoft Excel**

  * Pivot Tables
  * Pivot Charts
  * Slicers
  * Excel formulas
  * Data formatting

---

## 6️⃣ Methods Used

* Data cleaning and formatting in Excel
* Creating Pivot Tables for aggregating sales and profit data
* Analyzing product and regional performance
* Creating KPI metrics (Total Sales, Total Profit, Orders, Quantity)
* Designing an interactive dashboard using Pivot Charts and slicers

---

## 7️⃣ Key Insights

* **West region** generated the highest sales revenue
* **Electronics category** contributed the largest share of total sales
* **Laptop and Mobile** products were the top-performing items
* Monthly analysis revealed noticeable **sales growth during peak periods**
* Dashboard filters allow deeper exploration by **region, product, category, and month**

---

## 8️⃣ Steps to Run This Project

1. Download the dataset `sales_dashboard_dataset.xlsx`.
2. Open `Sales_Dashboard.xlsx` in Microsoft Excel.
3. Refresh Pivot Tables if necessary.
4. Use slicers to filter the dashboard by **Region, Product, Category, and Month**.
